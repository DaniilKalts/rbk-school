# Неделя 1 - Введение в Go

**Проект:** Текстовый процессор (CLI-инструмент)

## Требования

- Проект должен быть написан на `Go`
- Разрешено использовать только стандартные пакеты
- Нужно придерживаться принятых практик Go
- Структура проекта должна быть чистой и понятной

## Формат запуска

Программа должна принимать ровно 2 аргумента командной строки:

1. Путь к входному файлу
2. Путь к выходному файлу

Пример запуска:

```bash
go run . input.txt output.txt
```
